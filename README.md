# Thomas Strand Portfolio

Static portfolio website prepared for GitHub Pages.

## Publish

1. Extract the provided ZIP file.
2. In the `ttcstrand-oss/ttcstrand-oss.github.io` repository, choose **Add file → Upload files**.
3. Upload `index.html`, `styles.css`, and the complete `downloads` folder.
4. Commit the files to the `main` branch.
5. Open **Settings → Pages**.
6. Under **Build and deployment**, select **Deploy from a branch**.
7. Select the `main` branch and `/ (root)`, then save.

The site should publish at:

`https://ttcstrand-oss.github.io`
